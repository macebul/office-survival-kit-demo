### 6 Changes Approved & Implemented
1. **Booklets → Binder-clipped document sets** — Coil-bound booklets replaced with mixed-media document stacks held by oversized black binder clips. More authentic to "survivor assembled this" narrative, cheaper, mixed textures feel curated. Each set wrapped in "CASE FILE #XX — CLASSIFIED" paper band.
2. **Evidence bag grouping** — 4 items (Snitch Switch, Ankle Monitor, Evidence Eraser, Solitary) grouped into a single "CONFISCATED ITEMS" zip-lock bag with printed label. Simplifies Tray 2, saves packaging costs, the bag itself is a joke.
3. **Confiscated stapler empty recess** — One empty stapler-shaped chalk outline recess in Tray 2 with label "CONFISCATED — WEAPONIZED OFFICE EQUIPMENT. INCIDENT #147. DO NOT REISSUE." The absence IS the joke. $0 COGS.
4. **Yard Pass merged into Passport** — Corporate Parole Passport now contains yard pass pages (Break Room Access, Emergency Meeting Exit, Reply-All Immunity, Kevin Avoidance Permit, Bathroom Break). Passport page count increased to 12-16. Yard Pass eliminated as separate item.
5. **Die-cut foam → shredded "CLASSIFIED" paper** — Mug protection via crumpled printed paper resembling destroyed classified files. Triple duty: protects mug, extends joke, fills compartment visually. Thin foam backup sheet (~0.25") at bottom for drop protection.
6. **Mug COGS alternatives documented** — Stainless steel ($2-3 vs $4-6) and smaller 8oz size noted as fallback options. Enamel remains primary spec.

Enamel camp mug is the finale item revealed last. Target retail ~$24.99-29.99, COGS target under $10 (challenging).

- User prefers plan mode for non-trivial tasks
- Website is primarily for team visualization/alignment, not direct sales
- Amazon is primary sales channel
- Q4 2026 target for product-ready
- User values experience density over cost savings — would rather go "Deluxe+" than strip down

### Critical COGS Reality
Current design estimated at $15-22 COGS (per sourcing specialist analysis), not the $10 target. The 12 optimization/ideation changes help (thinner doc sets, shredded paper instead of foam, evidence bag consolidation, incident report notepad at same COGS, easter eggs at $0 COGS) but the fundamental tension remains: enamel mug ($4-6) + custom rigid box ($3-5) + tactical pen ($1.50-3) = $8.50-14 before any other items or assembly. Something will need to give when quotes come back.
