# Office Survival Kit — Documentation Roadmap

**Purpose:** Complete list of documents needed to take this product from concept to launch, organized by phase and priority.

**Version:** v6
**Last Updated:** March 17, 2026

**Architecture note (March 10, 2026):** Each document in this project has one specific job. No specs are duplicated across documents. If you need to know something, consult the authority document for that topic — see the Cross-Reference Map below.

**Major pivot note (March 13, 2026):** The product underwent a significant creative pivot. Key changes: (1) items reduced from 13 to 7 + 1 empty recess, (2) each tray item now has individual retail-style tuck-end boxes, (3) two-character narrative system introduced (OG creator vs. Corporate Nemesis who mass-produced the kit), (4) outer Amazon box added as outermost packaging layer, (5) Corporate Parole Passport replaced by Activity Guide (16-page saddle-stitched booklet), (6) evidence bag and guide cards removed, (7) OG sticky notes added as printed design elements on item boxes, (8) Pelican-style die-cut recesses retained but sized for product boxes. Five core documents updated: Item Specs v5→v6, Packaging Spec v8→v9, Dimensions Spec v9→v10, Copy Doc v5→v6, Documentation Roadmap v4→v5. Previous versions archived.
