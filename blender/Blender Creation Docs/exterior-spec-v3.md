# Office Survival Kit — Exterior Spec

**Last Updated:** March 11, 2026
**Status:** v3 — Dimension update pass. Supersedes v2 (March 10, 2026).
**Purpose:** Owns ALL exterior visual decisions — finish, materials, element placement, surface assignments, and element treatments for all six surfaces. Does NOT contain exact copy text (see Exterior Copy Doc v4) or interior details.

> **Dimensions reference:** All box dimensions, wall thicknesses, and derived measurements are owned by the Dimensions Spec. This document references those values for surface context only. If any measurement here conflicts with the Dimensions Spec, the Dimensions Spec wins.

---

## Overall First Impression

Somewhere between "serious" and "scrappy." The case reads as intentional and considered — not sloppy — but has clear improvised, contraband character. Like something assembled carefully under difficult conditions. The joke is that it looks like it means business. The details reveal it's homemade.

## 10. What Is NOT on the Exterior

- **No physical clasp hardware** — clasps are flat printed graphics only. Four magnets hidden behind/between them provide closure but are invisible from outside.
- **No panel lines** on any surface — no decorative panel borders, no embossed or printed panel divisions.
- **No decorative borders** or frame graphics on any surface.
- **No corner hardware** — no metal corner guards, no reinforced corners.
- **No glossy finish** on the box surface itself — box is fully matte. Applied elements (stickers, labels, stamps) get subtle satin finish only.
- **No color beyond:** matte black (box), cream/kraft (labels), red (rubber stamp), gray (duct tape handle), light gray/silver (printed clasp graphics), satin sticker colors on applied elements.
- **No rivets, screws, or applied metal** of any kind.
- **No corrugated cardboard** — solid board construction only.

## 13. Fit Flags

The following elements should be checked during layout/design for fit on the reduced-height faces:

| Flag | Surface | Concern |
|------|---------|---------|
| Nutrition Facts label height | Back face (14" × 5.0") | Standard FDA label format is tall. At 5.0" face height minus margins, the label may need to be narrower or the supporting labels arranged beside it rather than below. Confirm during design layout. |
| Front face element density | Front face (14" × 5.0") | Five elements (handle, 2 clasps, IT Asset Tag, INSPECTED BY, FRAGILE) on a 5.0"-tall face. Handle + clasps occupy the center band. Three labels must fit in the remaining space without crowding. Use full 14" width. |
| MONDAY vertical clearance | Left edge (11.5" × 5.0") | Diagonal placement on 5.0" height — letters may need to be sized to fit the shorter dimension. The wider 11.5" helps. |
