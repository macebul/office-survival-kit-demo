# Outer Amazon Box — Image Prompts

**Last Updated:** March 16, 2026
**Status:** Draft v1
**Purpose:** Image generation prompts for the outer Amazon box surfaces. Use these as starting points — refine based on output.
**Reference:** All copy from outer-box-copy-v1.md. All dimensions from amazon-strategy-v1.md.

**Dimensions:** Outer box is ~15.5" L × 13" W × 6.5" H (FBA Large Standard compliant).

**Critical aesthetic note:** This box is 100% Brad Sterling. NO institutional/prison aesthetic, NO duct tape, NO rubber stamps, NO worn textures — those belong to the briefcase inside. The outer box is Brad's vision of what a premium product looks like: polished FORMAT, unleashed CONTENT. The format (typography, layout, label designs, award seals) stays buttoned-up and corporate. The actual content goes for laughs on every line. Nobody buys a gag gift because they were fooled — they buy it because they laughed. Think: Apple meets WeWork meets a board game box that takes itself too seriously, written by a man with zero self-awareness.

**Box construction:** Tuck-top mailer box (one large panel flips open like a lid and tucks into the front). NOT a standard 4-flap shipping box. This gives one unbroken 15.5" × 13" interior lid surface for the welcome letter + Brad's photo. Opens like the briefcase inside — intentional echo. Confirm construction with sourcing.

**Brad's visual DNA (consistent across ALL photos):** Purple satin shirt (always), unbuttoned to mid-chest minimum, oversized gold rope chain — comically large, think 80s rapper, impossible to miss, aviator sunglasses, gold signet pinky ring, big cheesy teeth-too-white smile (always — infomercial host energy). Late 30s but dresses like a 22-year-old who watched Wolf of Wall Street once. Every photo was art-directed by Brad. He hired a photographer for each one. He thinks they're incredible. "PHOTO COURTESY OF ELEVATE CO." beneath every headshot.

## Generation Strategy

**Priority order for generation:**
1. **Prompt 1 (Hero shot)** — Generate first and test at thumbnail. This is the gatekeeper image. If this doesn't work at 150×150px, nothing else matters.
2. **Prompt 2 (Front face flat panel)** — The designer's primary reference for the #1 surface.
3. **Prompt 3 (Top face flat panel)** — The designer's primary reference for the #2 surface.
4. **Prompts 4-7** — Generate as needed for secondary surfaces.

**Thumbnail test protocol for Prompt 1:**
After generating, resize to 150×150px. Check:
- [ ] Can you read "OFFICE SURVIVAL KIT"?
- [ ] Can you see the gold award seal?
- [ ] Does it read as "premium gift box"?
- [ ] Is the Elevate Co. brand visible (even faintly)?

If any fail, adjust the prompt: increase product name size, simplify layout, push the award seal larger or brighter, reduce detail that becomes noise at small scale.

**Aspect ratio notes:**
- Prompt 1: --ar 1:1 (Amazon main image requirement)
- Prompt 2: --ar 15.5:6.5 (or approximately --ar 12:5)
- Prompt 3: --ar 15.5:13 (or approximately --ar 6:5)
- Prompt 4: --ar 15.5:6.5 (or approximately --ar 12:5)
- Prompt 5: --ar 15.5:13 (or approximately --ar 6:5)
- Prompt 6: --ar 13:6.5 (or approximately --ar 2:1)
- Prompt 7: --ar 15.5:13 (or approximately --ar 6:5)

**Known challenges:**
- AI image generators struggle with specific text. The product name "OFFICE SURVIVAL KIT" will likely need to be composited in post or hand-placed in the design file. Generate the layout and imagery, add text manually.
- The Nutrition Facts-style statistics banner may require separate generation and compositing.
- The testimonials on the back face will almost certainly need manual text placement.
- The gold award seal is a good candidate for a separate asset generation pass.
