# Office Survival Kit — Packaging Spec

**Last Updated:** March 3, 2026  
**Status:** Draft v3  
**Purpose:** Define the container, structure, and presentation for production.

---

## Concept: The Covert Attaché

The packaging is designed as a **professional attaché case** — the kind you'd see in a corporate setting — but when opened, it's laid out like a covert ops weapons case. Each "tool" has its own compartment. Lift one item, there's more underneath. It's a dead drop kit for surviving your corporate sentence.

**The joke:** It looks like something important is inside. Something serious disguised as something normal. Then you open it and discover... office supplies. But presented like they're mission-critical contraband.

- Fixed/glued to outer case (not removable)

## Interior Structure: Stacking Tray System

The interior uses a **stacking tray system** — each tray lifts out to reveal the next layer beneath. This maximizes copy space and creates multiple discovery moments.

### How It Works
- **Lid** opens to reveal first tray
- **5 trays** stack inside, each containing one item
- **Base** is the bottom of the case (closing copy)
- Each tray has a **printed lift tab** for easy removal
- Trays stack snugly to prevent shifting during shipping

### Item Silhouettes (Chalk Outline)
Each tray floor has a **printed silhouette/chalk outline** where the item sits:
- Evokes evidence scene / tactical case aesthetic
- Item rests ON TOP of the printed outline
- When item is lifted, outline remains visible (with hidden copy inside)
- Reinforces the "covert kit" concept without requiring foam die-cuts
- No additional production cost — it's printed artwork

### Tray Anatomy
```
┌─────────────────────────────────┐
│          LIFT TAB               │ ← Printed tab, centered
├─────────────────────────────────┤
│                                 │
│     ╔═══════════════╗           │
│     ║  CHALK OUTLINE ║          │ ← Printed silhouette of item
│     ║    + ITEM      ║          │ ← Item sits ON TOP of outline
│     ║ (HIDDEN COPY)  ║          │ ← Hidden copy inside outline
│     ╚═══════════════╝           │
│                                 │
│         VISIBLE COPY            │ ← Header, subheader, body
│                                 │
└─────────────────────────────────┘
        ↑ Tray walls (contain item)
```

## Lift Tab System

Each tray has a tab positioned in a **left-to-right cascade** — like manila folder tabs. Position indicates order (leftmost = first).

### Tab Layout
```
┌─────────────────────────────────────────┐
│ [SHANK] [SWITCH] [INK] [TAGS] [RAGS]   │
├─────────────────────────────────────────┤
│                TRAY 1                   │
└─────────────────────────────────────────┘
```

### Tab Copy

| Tray | Tab Copy | Position |
|------|----------|----------|
| Tray 1 | SHANK → | Far left |
| Tray 2 | SWITCH → | Left-center |
| Tray 3 | INK → | Center |
| Tray 4 | TAGS → | Right-center |
| Tray 5 | RAGS → | Far right |

**Tab design:**
- Size: ~1" wide × 0.4" tall
- Shape: Rectangular (institutional, file-folder feel)
- Position: Staggered left-to-right across tray width
- Print: White text on orange (#e85d04) background
- Material: Integrated with tray (folded paperboard)

## Copy Surfaces

Each tray has **two copy zones**:

### 1. Visible Copy (always visible)
- Printed on tray floor, around the item
- Contains: Header, subheader, body copy
- Position: Below or around item placement

### 2. Hidden Copy (revealed when item lifted)
- Printed on tray floor, underneath where item sits
- Short punchy line — reward for exploring
- Position: Centered under item footprint

### Lid Copy
- Full surface: 6.5" × 5"
- Contains: Product name, tagline, front cover design

### Base Copy
- Full surface: 6.5" × 5"
- Contains: Closing copy, branding, barcode/SKU area

## Reveal Sequence

The unboxing follows this order:

1. **Lid** — "OFFICE SURVIVAL KIT / Standard issue for your corporate sentence."
2. **Inside Lid** — Video game-style loadout/stats for all 5 items
3. **Tray 1** — Cubicle Shank (hero item, strong first impression)
4. **Tray 2** — Snitch Switch (contraband vibe builds)
5. **Tray 3** — Lifer Ink (earn your stripes) + Tattoo guide card
6. **Tray 4** — Turf Tags (claim territory) + Sticker guide card
7. **Tray 5** — Cry Rags (darkest joke, punchline position)
8. **Base** — "NOTICE" (fake warning label — works as back-of-box too)
