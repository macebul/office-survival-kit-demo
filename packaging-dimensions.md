## 6. Interior Structure: Stacking Tray System

The interior uses a stacking tray system. Each tray lifts out to reveal the next layer beneath, creating sequential discovery moments.

**Reveal order (top to bottom):**

1. Lid opens → Inside Lid (copy surface)
2. Tray 1 — Cubicle Shank
3. Tray 2 — Snitch Switch
4. Tray 3 — Lifer Ink + Tattoo Guide Card insert
5. Tray 4 — Turf Tags + Sticker Guide Card insert
6. Tray 5 — Cry Rags
7. Base (fixed — closing copy)

---

## 7. Tray Anatomy

Each tray follows this structure:

```
┌─────────────────────────────────┐
│          LIFT TAB               │  ← Orange tab, staggered L→R position
├─────────────────────────────────┤
│                                 │
│     ╔═══════════════╗           │
│     ║  CHALK OUTLINE ║          │  ← Printed silhouette of item (white/lt gray dashed)
│     ║    + ITEM      ║          │  ← Physical item rests ON TOP of outline
│     ║ (HIDDEN COPY)  ║          │  ← Hidden copy printed inside outline area (orange)
│     ╚═══════════════╝           │
│                                 │
│         VISIBLE COPY            │  ← Header (orange), subheader + body (white/lt gray)
│                                 │
└─────────────────────────────────┘
        ↑ Tray walls contain item
```

### Chalk Outline Style
- Simple filled silhouette of the item
- Color: White or light gray (#d0d0d0), dashed line border
- Item rests ON TOP of outline — outline remains visible when item removed
- Hidden copy printed inside the outline footprint, in orange (#e85d04)

### Copy Zones Per Tray
- **Visible copy** — printed on tray floor outside item footprint; always visible; contains header, subheader, body copy
- **Hidden copy** — printed on tray floor inside item footprint; concealed by item; revealed when item lifted; short single line, orange

## 11. Stack Height Calculation

| Component | Height |
|-----------|--------|
| Lid | 0.175" |
| Tray 1 walls | 0.700" |
| Tray 2 walls | 0.500" |
| Tray 3 walls | 0.500" |
| Tray 4 walls | 0.500" |
| Tray 5 walls | 0.700" |
| Base | 0.175" |
| **Total stacked height** | **3.250"** |
| Outer case depth | 3.375" |
| Remaining clearance | 0.125" |

ℹ️ **Note:** 0.125" of intentional clearance built in to account for paperboard caliper variation across five trays. Tray wall heights were not reduced — clearance is achieved by increasing case depth. Trays should still fit snugly with no significant shifting.

## 14. Typography & Visual Tone

| Element | Style Notes |
|---------|-------------|
| Headers | Bold, wide tracking, uppercase — institutional/military feel |
| Subheaders | Lighter weight, italic optional |
| Body copy | Small, clean, readable against dark backgrounds |
| Tab labels | Bold, uppercase, white on orange |
| Hidden copy | Italic, orange — reward/discovery feel |
| Exterior label | Slightly off-center placement, as if applied by hand |
| Rubber stamp | Distressed ink texture, slight rotation (~2–5°) |
| Barcode | Evidence/property-tag style |

**Overall tone:** Serious disguised as something normal. Bureaucratic and institutional on the outside; tactical and covert on the inside. No exclamation points. Dry, deadpan throughout.
